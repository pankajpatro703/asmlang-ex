# uProcessing
All microprocessor and microcontroller level computational codes in assembly and C.

* Processors
Extensions:     
    .asm       

* MCS-51
Extensions:        
    .a51    
    
* AVR 
Extensions:         
    .Sv8    
    .Sv16    
    .Sv32        

* PIC
Extensions:         
    .Sp12    
    .Sp14    
    .Sp16    
    .Sp18    
    .Sp24    
    .Sp32        

* ARM
Extensions:        
    .a4t  (7TDMI, 9TDMI)    
    .a6m  (M0, M0+, M1)    
    .a7m  (M3)    
    .a7em (M4, M7)    
    .a8m  (M23, M33)
